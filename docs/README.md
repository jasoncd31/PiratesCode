# Real Hot Girl Script Website
This repository hosts the website for Real Hot Girl Script, a programming language created by KeAnna Anglin, Lauren Lindsey, Shanaya Nagendran, and Sage Strieker for CMSI 488: Language Translation and Implementation at Loyola Marymount University.

This was made using a template and with reference to [https://dmoini.github.io/casper/](https://dmoini.github.io/casper/).

You can view the website at [https://laurenlindsey1.github.io/realhotgirlscript/](https://laurenlindsey1.github.io/realhotgirlscript/).
